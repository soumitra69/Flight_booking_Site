# ✈️ Flight Booking Website

Welcome to the official GitHub repository for the **Flight Booking Website** project. This is a modern, responsive travel booking frontend built with HTML, CSS, Bootstrap, and JavaScript.

🌐 **Live Preview**: [Visit the Website](https://soumitra69.github.io/Flight_booking_Site/)

---

## 📌 Features

- Sleek and modern user interface  
- Built using Bootstrap 5 for responsiveness  
- JavaScript-based dynamic sections (some in progress)  
- Simple navigation and layout  
- Booking form UI  
- Mobile-friendly design  

> 🚧 Some features are still under development. Stay tuned for updates!

- **CSS3**
